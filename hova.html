Hova

Micro framework for lightweight Golang web apps.

Hova is a micro framework to quickly create simple web applications in Golang. With a barebone structure, all Hova requires you to do is define routes and point them to views and actions, keeping development quick and easy.

Hova does everything for you. Params are readily available in your controller actions, variables are passed back to your views, Hova even handles templating for you! Three simple files, get started below.

Get started:

Pre reqs: Have your $GOPATH set properly.

1) Clone the repo and compile hova


        ➜ git clone https://github.com/coffeejay/hova.git
        ➜ cd hova_dir
        ➜ cd hova
        ➜ go install
        ➜ cd ../controllers
        ➜ go install
        ➜ cd ..


2) Open main.go and add a route, match it to a view, & define an action.


        package main

        import "hova_dir/hova"

        func main(){

        hova.AddRoute("/","views/index.html","Root")
        hova.Init()

        }


3) Open controller.go and write the action you just defined in main.go ("Root" in this case). Accept params in the header and return the variables you want available in the corresponding view.


        package controller

        type Controller struct{}

        func (c *Controller) Root(params map[string][]string) map[string] string{
          greeting := "Hello, " + params["name"][0] + "!"
          return map[string]string{"greeting":greeting} 
        }


4) Ready your view accordingly.


          < h1>
          {{.greeting}}
          < /h1>


5) Go look at http://localhost:9000, you're done!

For full documentation and all features, keep going!


Hova's file structure. You must run "go install" in both the hova directory and controllers directory to compile the files (and in order for Hova to work).


          hova_dir/
            main.go
            views/
            hova/
              hova.go
            conrollers/
              controller.go

Let's break down the files:


main.go


          Use Hova.AddRoute(route, view, action string) to give Hova routes.
          ➜ View names can be relative paths ("views/index.html")

          When you've added all routes, initialize Hova by calling Hova.Init()
          ➜ Hova will start listening on port 9000.

controller.go


          Define and write your actions in this file. 

          ➜ Controller actions must take the Controller struct as a recevier.
          ➜ Accept params as a map[string][]string for access to params 
          received in the call.
          ➜ Return any variables in your actions that you want access to
          in the view in a map[string]string.

          func (c *Controller) SomeFunc(params map[string][]string){
                name := "roshamboe"
                return map[string]string{"name":name}
          }

That's it, it's that simple. Go try it now!
